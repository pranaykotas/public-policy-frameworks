# Frameworks for Public Policy

**[frameworks.pranaykotas.com](https://frameworks.pranaykotas.com)**

99 public policy frameworks — each explained in plain language with its origin, how to apply it, where it breaks down, and links to the original sources. Drawn from 300+ editions of [*Anticipating the Unintended*](https://publicpolicy.substack.com), a newsletter on Indian public policy.

Useful if you are a student building a policy vocabulary, a practitioner looking for a framework to apply to a problem on your desk, or a curious reader who wants to understand how policy thinkers approach the world.

## Suggest a framework

Know a framework that belongs here? [Open an issue](https://github.com/pranaykotas/public-policy-frameworks/issues/new?title=Framework+suggestion&body=Framework+name:%0A%0AWhere+it+comes+from:%0A%0AWhy+it+belongs+here:) with the name, where it comes from, and why it's useful. Good ones get added.

## Built with

[Quarto](https://quarto.org) — one `.qmd` file per framework, deployed via GitHub Actions to GitHub Pages.
